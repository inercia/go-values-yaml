# go-values-yaml

Golang library skeleton. Implementation will be added later.

## Development

- Requires Go >= 1.22
- Common commands:
  - Build: `make build`
  - Test: `make test`
  - Lint: `make lint`
  - Format: `make fmt`
  - CI checks: `make ci`

## Module path

The default module path is `github.com/inercia/go-values-yaml`. Update `go.mod` if your GitHub org/repo differ.
